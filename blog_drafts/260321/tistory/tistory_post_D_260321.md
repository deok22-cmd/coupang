## 1. 개요
일반적인 수평형 마우스 사용 시 발생하는 전완근의 회내(Pronation) 현상은 요골과 척골 사이의 근막 긴장도를 높여 장기적으로 정중신경 압박에 의한 수근관 증후군(Carpal Tunnel Syndrome)을 유발할 수 있음. 이에 따라 손목을 수직 방향(Vertical)으로 유지하여 연부 조직의 부하를 분산시키는 인체공학적 입력 장치의 도입이 직업적 건강 관리에 매우 중요함.

## 4. 손목 건강을 위한 올바른 입력 장치 사용 가이드
1.  **적절한 각도 유지**: 버티컬 마우스 사용 시 팔꿈치를 책상 위에 안정적으로 거치하고 마우스를 마치 '악수하듯' 가볍게 쥐었을 때 가장 이상적인 근육 이완 상태가 됨.
2.  **센서 감도 최적화**: 마우스 이동 거리가 과도할 경우 어깨에 무리가 갈 수 있으므로, DPI를 적절히 높여 손목의 가동 범위를 최소화함이 권장됨.
