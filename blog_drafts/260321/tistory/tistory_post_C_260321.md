## 4. 데이터 기반 벨벳 거품(Velvet Foam) 생성 노하우
1.  **우유 온도 제어**: 우유 내 단백질이 안정적으로 팽창하는 60~65℃ 구간에서 거품 생성이 가장 안정적이며, 70℃ 이상 가열 시에는 기포가 쉽게 파괴됨.
2.  **헤드 위치 선정**: 휘핑 헤드를 용기 바닥이 아닌 액체 표면 살짝 아래에 45도 각도로 배치할 때 공기 유입량이 최대화됨.
3.  **세척 알고리즘**: 우유 단백질의 변성을 방지하기 위해 사용 직후 흐르는 정수(Water) 아래에서 모터를 짧게 구동 시키는 것이 가장 효율적인 관리법임.
